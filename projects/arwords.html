Arwords

A mobile app for Lebanese Arabic learners

The Concept

Arwords aka Arabic Words. We need an app that helps to quickly find a word in Lebanese dialect. Current transalators like Google Translator use the official arabic dialect, so pretty much useless. Also, the app must be working offline too.

Implementation

Using flutter after so much battle with expo and react native js libraries issues.

Result

Currently in closed testing in Google Play Store
